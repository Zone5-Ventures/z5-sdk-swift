FileUpload README

Include the Zone5 SDK using Swift Package Manager:

    File -> Swift Packages -> Add Package Dependency
    Enter this URL: "https://github.com/Zone5-Cloud/z5-sdk-swift.git"

Import Zone5 SDK where needed in your Swift file:

    import Zone5
    
Set login credentials:

    set the hardcoded constant values in ViewController.swift

Run this demo app to see how to create a new login, and forgot password
